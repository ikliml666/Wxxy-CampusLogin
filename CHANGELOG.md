# Changelog

## v2.2.4 (2026-06-08)

### 修复

- **无网络时误触发校园网退出**: 当配置的适配器均未获取IP（完全无网络连接）时，校园网检测会错误判定为"非校园网"并触发自动退出。现在增加无IP判断，无网络时跳过退出流程，等待网络恢复后重新检测

### 新增

- **适配器级获取新IP**: 新增 `dhcp_release_renew_adapter` 命令，支持指定单个适配器修改MAC获取新IP
- **首页快捷操作改进**: "获取新IP"按钮标注为"获取新IP(主)"，明确操作主适配器
- **双适配器下拉选择**: 启用双适配器模式时，首页"获取新IP"按钮悬停弹出适配器选择菜单，可选主/副适配器
- **网络适配器面板操作按钮**: 每个适配器卡片添加"获取新IP"按钮，支持对单个适配器执行MAC切换

### 变更

- **"备用适配器"重命名为"副适配器"**: 所有界面文本中"备用适配器"改为"副适配器"，英文"Backup Adapter"改为"Secondary Adapter"
- **适配器列表排序**: 网络适配器面板中主适配器排在最上方，副适配器次之
- **获取新IP按钮样式**: 网络适配器面板中的"获取新IP"按钮改为 outline 样式，添加琥珀色边框以区分背景
